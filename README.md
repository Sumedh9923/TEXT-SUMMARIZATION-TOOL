## ğŸ“Œ Overview
The **Text Summarization Tool** is a Natural Language Processing (NLP) based Python application designed to generate concise summaries from lengthy articles or paragraphs.  
In the modern digital world, massive amounts of textual data are generated daily in the form of news articles, blogs, academic papers, reports, and documentation. Reading full-length content can be time-consuming and inefficient. This tool solves that problem by automatically identifying the most important sentences in a text and producing a meaningful summary while preserving the original context.
This project implements **Extractive Text Summarization**, where key sentences are selected based on statistical importance rather than generating new sentences.
